# Explanation

  C supports the varieties of integeral data types declared using `int` `long` `double` and more... Some could only represent
  non-negative numbers and some could represent negative,positive and zeroes.They have ranges till which they could represent
  numbers.On 64-bit operating systems they use an 8byte reperesentation and on the 32-bit operating system they use 4byte
  representation.
